# Quotes Scraper

Un script en Python pour récupérer automatiquement des citations sur le site "Quotes to Scrape"[cite: 1].

## Fonctionnalités
* Scrape les 5 premières pages du site[cite: 1].
* Extrait le texte de la citation et le nom de l'auteur[cite: 1].
* Affiche les résultats dans la console[cite: 1].
* Sauvegarde automatiquement toutes les données dans un fichier `result.csv`[cite: 1].

## Technologies utilisées
* Python 3
* Requests (pour les requêtes HTTP)
* BeautifulSoup4 (pour le parsing HTML)
* CSV (bibliothèque native)
